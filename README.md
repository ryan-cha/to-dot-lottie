# ryan-to-dot-lottie

## What's different?

- Removed the check on the path for presence of specific folder name.
- Exit the process (with result `0`) once the job is done.

## How to build and run

```sh
# generate binary
./gradlew distZip

# unzip the distribution file
$ cd lib/build/distribution
$ unzip -d . lib.zip

# run the binary
$ ./bin/lib
```
